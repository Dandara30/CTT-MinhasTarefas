<h1 align="center">Projeto Minhas Tarefas</h1>
<p align="left"><b>Objetivo:</b> Aplicativo para organizar tarefas do dia-a-dia.</p>


### Features:

- <b>Listagem das tarefas</b>: Visualização das tarefas via listagem onde serão carregadas pelos status de: A Fazer, Em Progresso e Feitas.
- <b>Cadastrar Tarefa</b>: Cadastro de novas tarefas, onde é possível incluir Título e uma breve descrição.
- <b>Editar Tarefa</b>: Alteração das informações atuais da tarefa antes de finaliza-las.
- <b>Excluir Tarefa</b>: Ação de Apagar a tarefa em qualquer status que ela se encontra.

- <b>Iniciar Tarefa: Ao clicar em uma "Tarefa a Fazer é possível Iniciar Tarefa, onde ela será visualizada apenas na lista do status <b>Em Progresso</b>.
- <b>Finalizar Tarefa: Ao clicar em uma "Tarefa a Fazer é possível Finalizar Tarefa, onde ela será visualizada apenas na lista do status <b>Feitas</b>.


### Conceitos/Tecnologias utilizados:

- Kotlin
- Conceitos básicos de MVVM
- Conceito básicos de Room
- ConstraintLayout
- Bottom Sheet Dialog
- RecyclerView / Adapters
- ViewPager e TabLayout
